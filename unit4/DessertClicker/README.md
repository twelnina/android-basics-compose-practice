# Dessert Clicker

Dessert Clicker is a simple clicker game built as a practice project for Android Basics with Compose.

Tap the dessert in the center of the screen to increase the number of desserts sold and total revenue. The displayed dessert changes as the sales count grows. This project focuses on basic Jetpack Compose UI implementation, state management, and observing the Android Activity lifecycle.

## What I Practiced

- Building screens with Jetpack Compose
- Managing UI state with `ViewModel` and `StateFlow`
- Using Compose components such as `Scaffold`, `TopAppBar`, `Image`, and `Text`
- Observing Activity lifecycle methods with Logcat
- Working with resources, strings, and themes

## Features

- Tap a dessert image to increase the sales count
- Calculate total revenue based on each dessert's price
- Change the displayed dessert based on the number of desserts sold
- Display sales count and revenue in the top app bar
- Log Activity lifecycle events

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- AndroidX Lifecycle / ViewModel
- Gradle Kotlin DSL
